== Description ==

Displays Github activity in a fancy way via shortcode which can be used in posts, pages, or widgets.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fancy-github-activity` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use shortcode [fancy-github-activity] where you would like it. Configurable options below.

Options:

* username="stopspazzing" // required
* repository="" // optional
* limit="20" // optional

Example:
<code>[fancy-github-activity username="stopspazzing"]</code>
